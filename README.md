TASK-1

Description: Write a program to manage a file of words or dictionary.
Text File name: dictionary.txt

Objective:
	i.   Add new words
	ii.  Search for list of words by entering some characters
	iii. Change spelling of a word
	iv.  Delete a word
	v.   Remove duplicate words
	vi.  Arrange and print the words alphabetically
